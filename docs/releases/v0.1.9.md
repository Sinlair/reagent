# ReAgent v0.1.9

Single-package release.

## Highlights

- one official package
  - `@sinlair/reagent` is now the only public install target
- repo cleanup
  - removed internal package directories and aligned the repo around the standalone runtime
- clearer compatibility messaging
  - OpenClaw stays a reference and integration direction, not a second ReAgent package
- cleaner release flow
  - verification, packing, and publishing now target the root package only

## Package

- `@sinlair/reagent`
  - standalone always-on runtime and CLI

## Validation

Validated for this release with:

- `npm.cmd run release:verify`
- `npm.cmd run release:pack`
- `node --check web/app.js`

## Recommended Release Notes Summary

ReAgent v0.1.9 completes the move to a single-package product. The repo, docs, release flow, and user-facing runtime surfaces now all point to one official install target: `@sinlair/reagent`.
