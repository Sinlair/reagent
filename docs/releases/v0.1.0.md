# ReAgent v0.1.0

Initial public release of ReAgent as a local research workspace.

## Highlights

- multi-surface web workspace
  - Home
  - Command Center
  - Agent Desk
  - Evidence Workspace
  - Research Map
  - Knowledge Vault
  - Channels
- structured `Research Brief`s with markdown import/export
- brief-driven discovery and ranking
- paper analysis, repo analysis, and module extraction
- baseline suggestion and direction reports
- meeting deck generation
- feedback loop and memory graph
- WeChat / OpenClaw runtime integration
- always-on runtime support
  - PM2
  - Windows service scripts
  - scheduler visibility
  - lifecycle audit visibility

## Packages

- root app
  - local web workspace and full runtime
- `package/`
  - in-repo OpenClaw WeChat reference package kept for compatibility and integration work

## Quick Start

```powershell
Copy-Item .env.example .env
npm.cmd install
npm.cmd run db:push
npm.cmd run dev
```

Minimal local config:

```env
LLM_PROVIDER=fallback
WECHAT_PROVIDER=mock
```

Open:

- `http://127.0.0.1:3000/`

## Always-On Runtime

PM2:

```powershell
npm.cmd run pm2:start
```

Windows service:

```powershell
npm.cmd run service:install
```

## OpenClaw Compatibility

At the time of this release, OpenClaw compatibility was tracked in-repo as
reference integration work rather than a second official ReAgent install path.

## Validation

Validated before the public release with:

- `npm.cmd run build`
- `npm.cmd run check:all`
- `npm.cmd run test`

## Known Limitations

- the root app remains `private: true` and is not intended to be published directly to npm
- the `package/` reference package is kept in-repo and may be confusing without the README repo-layout section
- scheduler and lifecycle views are usable, but not yet a full historical operations console
- some deeper roadmap items are still pending
  - richer cron/job controls
  - more complete artifact browsing
  - broader package/plugin parity
  - deeper multi-agent orchestration

## Recommended Release Notes Summary

ReAgent v0.1.0 is the first public release of a local research workspace focused on continuous research workflows rather than one-shot chat. It combines structured research briefs, discovery, evidence synthesis, memory, reports, decks, and WeChat/OpenClaw integration into one product surface, with support for always-on runtime operation through PM2 and Windows service scripts.
