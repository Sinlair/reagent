# Research Brief

When the user asks for analysis, comparison, synthesis, or a recommendation:

- prefer evidence-backed claims
- separate observed facts from inference
- call out uncertainty when information is incomplete
- avoid long generic prose when a compact brief is enough

If a research run already exists, prefer summarizing the existing result before launching a new one.
